# Youth Program Data Collection & Analysis Project

## Overview
This project simulates a REDCap-style data collection and analysis system used in research and nonprofit settings.

## Objectives
- Design survey-based data collection tools
- Ensure data integrity and validation
- Analyze participant outcomes using SQL
- Generate insights to evaluate program effectiveness

## Tools Used
- Google Forms (survey design)
- Excel / CSV (data storage)
- SQL (data analysis)
- Tableau (optional visualization)

## Key Features
- Survey design with validation and branching logic
- Structured dataset for participant tracking
- SQL queries for outcome analysis
- Data cleaning and validation processes

## Key Insights
- Average participant improvement increased post-program
- High-risk participants showed measurable progress
- Follow-up scores indicate retention of improvements

## Skills Demonstrated
- Data collection design
- Research methodology
- SQL querying
- Data validation and integrity
- Analytical reporting

## Data Collection Forms

I designed two structured Google Forms to mimic real-world research data collection:

### 1. Youth Intake Assessment
Captures baseline participant information including:
- Demographics
- Risk level
- Program eligibility factors

### 2. Outcome Assessment (Pre / Post / Follow-Up)
Captures participant outcomes over time:
- Pre-program scores
- Post-program scores
- Follow-up scores
